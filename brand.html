CCA Brand Guide

This page provides principles and assets to aid in presenting a unified visual brand for Columbus Classical Academy.

Colors

CCA's primary colors are Blue and Red. White is used as needed for contrast.

▬▬▬▬▬ CCA Blue

  • Hex: #003A70
  • RGB: (0, 58, 112)

▬▬▬▬▬ CCA Red

  • Hex: #ED1E24
  • RGB: (237, 30, 36)

▬▬▬▬▬ White

  • Hex: #FFF
  • RGB: (255, 255, 255)

Typeface

Serif Typeface: Libre Baskerville

  • The serif typeface is the primary typeface to be used on all assets including web, print, signage, etc.
  • Link: https://fonts.google.com/specimen/Libre+Baskerville
  • CSS Import: @import url('https://fonts.googleapis.com/css2?family=DM+Sans&family=Libre+Baskerville&display=swap');
  • CSS: font-family: 'Libre Baskerville', serif;

Sans-Serif Typeface: DM Sans

  • The sans-serif typeface is an alternate typeface to be used where required, primarily as body text on web.
  • Link: https://fonts.google.com/specimen/DM+Sans
  • CSS Import: @import url('https://fonts.googleapis.com/css2?family=DM+Sans&family=Libre+Baskerville&display=swap');
  • CSS: font-family: 'DM Sans', sans-serif;
